## ðŸš€ Getting Started Locally

### 1. Prerequisites
Ensure you have the following installed:
- Node.js (v20 or higher)
- pnpm (v8 or higher)
- Docker (for database and previewing)

### 2. Installation
Install all dependencies from the root directory:
```bash
pnpm install
```

### 3. Environment Setup
Create a `.env` file in the root or in specific packages as needed. The Foundry platform requires a PostgreSQL database.

You can start a local PostgreSQL instance using Docker:
```bash
docker-compose up -d
```

### 4. Running the Foundry
You can run the entire platform (Web + API) using a single command:
```bash
pnpm dev
```
- **Web UI**: `http://localhost:3000`
- **Backend API**: `http://localhost:3001/api`

### 5. Generating an App
1. Open the Web UI.
2. Enter your startup pitch in the "Project Wizard".
3. Review the AI-generated Blueprint.
4. Click **Build & Deploy**.
5. The generated code will appear in the `generated/` directory.
